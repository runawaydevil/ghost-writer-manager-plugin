# Templates de exemplo (pt-BR)

Arquivos **Markdown** de referência. Eles **não** são carregados automaticamente pelo plugin: copie o conteúdo para uma nota dentro da **pasta de sincronização** do seu vault (ou use o comando **Create new Ghost post** no Obsidian).

## Prefixo

Os exemplos usam o prefixo padrão do código: **`ghost_`**. Se nas configurações do plugin você usar outro prefixo (ex.: `g_`), renomeie todas as chaves do frontmatter.

## Arquivos

| Arquivo | Cenário |
|---------|---------|
| `post-rascunho.md` | Rascunho no Ghost, visibilidade paga (ajuste `ghost_post_access` se quiser público). |
| `post-publico.md` | Post público, publicação imediata. |
| `post-agendado.md` | Publicação agendada (altere a data ISO). |
| `post-com-paywall.md` | Prévia pública + conteúdo só para membros/assinantes. |
| `post-destaque-tags.md` | Destaque no site, tags e resumo customizado. |
