# Ghost Writer Manager — documentação (pt-BR)

Material em **português do Brasil** para uso do plugin [Ghost Writer Manager](https://github.com/diegoeis/ghost-writer-manager-plugin) no Obsidian.

| Arquivo / pasta | Descrição |
|-----------------|-----------|
| [instrucoes-de-uso.md](./instrucoes-de-uso.md) | Instalação, configuração, comandos, frontmatter, calendário e paywall |
| [templates/](./templates/) | Modelos de nota (`.md`) prontos para copiar ou adaptar |

## Observação sobre o prefixo YAML

No código do plugin, o prefixo padrão das propriedades Ghost no frontmatter é **`ghost_`** (por exemplo, `ghost_published`). Se você alterar **Prefixo YAML** nas configurações do plugin, substitua `ghost_` por esse prefixo em todos os exemplos e templates desta pasta.
